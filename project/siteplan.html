Learning Techniques Site Plan

Name

Learning Techniques

Teaching is the best way to learn is to teach said Frank Oppenheimer. This website is intended to be used as a tool for adults that are interested in teaching, want to improve their teaching skills, or want to understand the teaching methods of their teachers. I intend to show the best proven techniques and create a link from those techniques to the ones used by the Savior. That way the teachers can become like the ultimate teacher, and build their students confidence and skills.

Purpose

Learning Techniques exposes proven methods of teaching and learning. The site has:

  • Teachniques descriptions
  • Case Studies
  • Examples on class preparation
  • Quotes from the greatest thinkers in the discipline
  • Video Links
  • Quizzes
  • Checklists

Scenarios

  • How can I teach different kinds of people?
  • How can I know if my teaching was effective?
  • How can I create trust in the class so students can feel confident making questions?
  • Which tools can be used to make the class more dinamic?
